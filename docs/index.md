# all9s Solutions Eclipse Tools

all9s Solutions Eclipse Tools provide small Java editor productivity plug-ins for reducing repetitive source-editing work.

## Tools

- [Ender](ender.md) adds ending comments to Java blocks.
- [Lumberjack](lumberjack.md) generates Java logging statements.

## Install

For local testing, install from `file:/D:/Workspaces/projects/com.all9ssolutions.update/`. For public use, publish the update site to HTTPS and use that URL in Eclipse Marketplace.
