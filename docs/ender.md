# Ender

Ender adds configurable ending comments to Java methods and control blocks.

## Quick Start

1. Open a Java source file.
2. Run Ender from the toolbar, editor context menu, or `Ctrl+Shift+E`.
3. Review the generated comments around nested blocks.
4. Change comment style in `Window > Preferences > General > Ender Preferences`.

## Examples

Ender can add comments such as `// end if`, `// end for`, and `// end try/catch` without replacing existing custom ending comments.
