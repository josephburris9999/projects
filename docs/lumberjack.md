# Lumberjack

Lumberjack generates configurable Java logging statements for methods, parameters, return values, and common control-flow statements.

## Quick Start

1. Open a Java source file.
2. Run Lumberjack from the toolbar or editor context menu.
3. Choose logging options in the Lumberjack preferences.
4. Review generated statements before committing code.

## Examples

Use Lumberjack to add method-entry, parameter, return-value, and branch-oriented logging statements to existing Java code.
