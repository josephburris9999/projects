The suite

Streamlined commands for faster Java coding.

The feature packages Ender, Lumberjack, the Lumberjack help bundle, and shared branding into an Eclipse p2 update site. The plug-ins activate in the Java editor and expose commands through the main toolbar, editor context menu, and keyboard bindings.

Ender

Adds configurable ending comments to Java methods and control blocks so long files stay easier to scan. Preferences let teams choose the ending-comment style they want generated.

  • Command shortcut: Ctrl+Shift+E
  • Workbench preference page
  • Available in Java editor scope

Lumberjack

Opens a wizard that previews generated logging before applying changes. It supports framework choices, method entry and exit logging, control-flow statements, variables, method calls, and switch statements.

  • Command shortcut: Ctrl+Shift+L
  • Selections and preview wizard
  • Bundled Eclipse help content
