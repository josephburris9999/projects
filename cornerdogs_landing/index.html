Wizard — Form Wizard Using J2EE & Custom Tags

At several points in my career I've built 'wizard' style sites — requesting information from clients via rolling pages, each page focusing on a specific area of interest. With each site, no matter the supporting framework, my design evolved into a simple, stream-lined template.

Client side, the site provides convenient features:

  • Pages are navigable with next and previous buttons.
  • Validation is available for required questions.
  • Breadcrumbs are available for page history.
  • Modern Bootstrap front-end for responsive HTML.

Structurally, the site relies on session management and a strong MVC pattern:

  • Robust Objects with comprehensive instance methods.
  • Fragments and custom tags for common information in the views.
  • Simple controllers managing logic flow and traffic.
  • Absent persistence strategy allowing for customization.

This version of the template is written with J2EE APIs for adaptiblility to a Java-based framework. If you find it useful, tell me about it!
