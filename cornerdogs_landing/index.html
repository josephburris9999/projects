Corner Dogs

A focused hot dog ordering demo built to demonstrate a practical multi-step web wizard: customer intake, order details, verification, validation, session flow, and a clean Tomcat 11 deployment path.

Guided Flow

Moves visitors through focused pages with next, previous, cancel, and verification actions.

Validated Forms

Checks names, quantities, order location, and confirmation choices before continuing.

Modern Runtime

Runs as a Jakarta Servlet/JSP application packaged for Apache Tomcat 11.

A Small App With Real Web Patterns

Corner Dogs turns a fictional hot dog order into a compact demonstration of server-rendered Java web application structure. The order flow collects a customer name, asks where the order will be taken, captures hot dog quantities, calculates totals, and then presents the visitor with a final verification step.

The submit action is intentionally demo-only. No order is persisted, emailed, charged, or sent to a restaurant system, so the project stays safe to explore while still showing how the pieces fit together.

Built For Clear Demonstration

The project shows a traditional MVC-style servlet application with JSP views, reusable fragments, custom form tags, typed model objects, breadcrumbs, and Log4j-backed runtime logging. Maven owns the repeatable build, while Eclipse Dynamic Web Project metadata remains available for local development.

Runtime Java 17 or newer, Jakarta Servlet 6.1, JSP 4.0, Apache Tomcat 11, and Log4j 2.26.0.

What To Look For

Corner Dogs is intentionally modest in scope, which makes the architecture easy to inspect. It is a useful reference for validating form pages, carrying state through a session-backed wizard, and keeping the final submit path honest about what the demo does and does not do.

Servlet Controllers

Page-specific servlets coordinate navigation and model updates while keeping JSPs focused on rendering.

Custom Tags

Reusable tags render common form controls, making validation messages and field markup consistent.

Maven Packaging

The app builds into target/cornerdogs.war for repeatable deployment to Tomcat.

Corner Dogs Project Details

Corner Dogs is a portfolio demonstration application by Joseph Burris, all9s Solutions LLC. It was migrated from an older Java EE / Java 8 Dynamic Web Project to Jakarta EE for Tomcat 11.

The app demonstrates wizard navigation, JSP rendering, servlet controllers, custom form tags, session-backed state, validation, and demo-only order submission.
