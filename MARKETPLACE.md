## Listing Copy

Ender adds configurable ending comments to Java methods and control blocks so nested Java source is easier to scan.

Lumberjack generates configurable Java logging statements for methods, parameters, return values, and common control-flow statements.

The complete suite installs both tools with shared all9s Solutions branding.
